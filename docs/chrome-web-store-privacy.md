# Chrome Web Store Privacy Draft

Use this text when filling out the Chrome Web Store Privacy tab.

Public privacy policy URL for Chrome Web Store:

<https://huruikagi.github.io/table-enhancer-for-github/privacy-policy/>

## Single Purpose

Table Enhancer for GitHub improves readability and navigation for wide tables on GitHub Markdown blob file preview pages.

## Permission Justification

### storage

Used only to save the user's heading-based freeze defaults locally in Chrome storage. The extension uses these settings to restore table freeze defaults on matching GitHub Markdown blob pages.

The extension does not sync, transmit, sell, or share this data.

## User Data Handling

This extension does not collect, transmit, sell, or share user data.

It stores table display preferences locally in the user's browser using `chrome.storage.local`. These preferences are used only by the extension to restore the user's table freeze defaults on matching GitHub Markdown blob pages.

## Remote Code And Network

The extension does not load remote code and does not send network requests.

## Privacy Policy Summary

Table Enhancer for GitHub does not collect, transmit, sell, or share user data. It stores table display preferences locally in the user's browser using Chrome local storage so the extension can restore heading-based freeze defaults on matching GitHub Markdown blob pages.
